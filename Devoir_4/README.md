# Devoir 4 - Filiation de processus sous la forme d'un en ensemble de processus

### Ecrire un programme un programme en langage C qui génère dynamiquement une hiérarchie de processus organisée sous forme d'un arbre binaire complet.   Le programme doit créer un ensemble de processus dont la filiation forme un Chaque processus parent crée exactement deux processus fils

### Nombre de niveaux de l'arbre à partir de la racine (niveau 0)  , où K est fourni en argument sur la ligne de commande.  

Exemple avec K = 2 (profondeur 2) :

         P0 (racine)
        /  \
      P1    P2        (niveau 1)
     / \    / \
   P3  P4  P5  P6     (niveau 2)



Exemple d'execution avec K=10 .
./processarbre 10
